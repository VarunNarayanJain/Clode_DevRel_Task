# Competitor Research

This section contains detailed research on selected companies operating in the cloud infrastructure, DevOps, platform engineering, and adjacent spaces.

Each company is analyzed with respect to:
- What problem they solve
- Who they are built for
- How they position themselves
- Where they overlap with Clode
- How Clode differentiates itself

Each competitor is documented in an individual markdown file.
